CSS Specificity

CSS Specificity คืออะไร

Css Specificity คือค่าความสำคัญที่จะเป็นตัวกำหนดให้ browser เลือกใช้ style ในการแสดงออกมาทางหน้าเว็บ ยิ่ง style นั้นมีความเจาะจงมากเท่าไร ก็ยิ่งมีความสำคัญมากขึ้นเท่านั้น ถ้าเราเข้าใจ Css Specificity ก็จะช่วยให้หน้าเว็บของเราแสดงผลตามที่เราต้องการ


ประโยชน์ของ Css Specificity

1.ช่วยให้เราเข้าใจว่าทำไมบางครั้ง stye ที่เราเขียนไม่แสดงผลออกมา
2.ช่วยจัดระเบียบให้กับ css code ของเรา
3.ช่วยให้เราเขียน code น้อยลง และดู clean ขึ้น


หลักการทำงาน

โดยเราสามารถแบ่งออกได้เป็น 3 กลุ่มใหญ่ๆ คือ
1.Type selectors & pseudo-elements
2.Class selectors , attribute selectors และ pseudo classes
3.ID selectors


กฎเพิ่มเติม

1.Universal selectors, combinators เช่น + , - ไม่มีผลกับ Css Specificity
2.Inline styles ไม่ค่อยเป็นที่นิยมในการเขียน ควรที่จะเขียนใน .css แยกต่างหาก
3.!important ถ้าไม่จำเป็นจริงๆ ไม่ควรใช้เพราะเป็นการบังคับให้ใช้ style นั้น เมื่อเราเขียนแบบนี้ไปเรื่อยๆอาจทำให้ยากต่อการ debug


การคำนวณ

จากรูปเรากำหนดให้มีค่าเป็น(0 , 0 , 0 ,0) ถ้ามี element ตัวไหนก็ให้แก้ไขค่าจาก 0 --> 1 เช่น .glass{ font-size: 27px; } จะมีค่าเป็น (0 , 0 , 1 , 0) เป็นต้น
